# TP Git - Initiation à Git et GitHub

## Description

Projet d'initiation aux outils de versioning Git et à la plateforme GitHub. Ce TP met en pratique les commandes de base et le workflow Git avec un petit projet web combinant HTML, CSS et PHP.

## Fonctionnalités

- Page HTML simple avec affichage de la date via PHP
- Stylisation CSS de base
- Démonstration de l'intégration Git/VSCode

## Technologies utilisées

- HTML5
- CSS3
- PHP
- Git & GitHub
- Visual Studio Code

## Structure du projet

```
tp_git/
├── Ex1.php        # Page principale avec affichage de date
├── styles.css     # Feuille de style
└── README.md      # Documentation
```

## Installation

1. Cloner le repository :
```bash
git clone https://github.com/Lelio88/tp_git.git
```

2. S'assurer d'avoir PHP installé sur votre machine

3. Lancer le fichier avec un serveur PHP :
```bash
php -S localhost:8000
```

4. Ouvrir dans le navigateur : `http://localhost:8000/Ex1.php`

## Objectif pédagogique

Ce projet a été réalisé dans le cadre d'un apprentissage des fondamentaux de Git et GitHub, incluant :
- Initialisation d'un repository
- Commits et historique
- Push vers un repository distant
- Utilisation de VSCode comme environnement de développement
